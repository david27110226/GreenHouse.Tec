Manual de usuario

El manual proporciona información detallada sobre la funcionalidad y el uso de la página web de Greeen House Tec, que consta de varias secciones clave. En primer lugar, presenta una página de inicio de sesión y registro para que los usuarios accedan al sistema. Una vez dentro, los usuarios son dirigidos a la página principal, que incluye un menú de navegación que les permite acceder a diferentes secciones. Entre estas secciones se encuentra la base de datos, que almacena información del invernadero automatizado. También se encuentran secciones dedicadas a los mantenimientos, manuales y un apartado de contacto que incluye un chatbox para que los usuarios puedan enviar quejas o sugerencias. Este manual proporciona una guía detallada sobre cómo navegar por la página web, acceder a la información relevante y utilizar las funciones disponibles de manera efectiva.
